## About

Simple CMS on PHP.

### Built With

- [PHP](https://www.php.net/)
- [Bootstrap](https://getbootstrap.com)

## Getting Started

### Prerequisites

- npm
  ```sh
  composer update
  ```

### Installation

1. Clone the repo

```sh
git clone https://github.com/developer01234/CMS-PHP.git
```

2. Start server

```sh
php -S localhost:8000
```

or Apache

3. Configure db

Example /config/database.php

```

<?php
return [
    'host'     => 'localhost',
    'db_name'  => 'cms',
    'username' => 'root',
    'password' => '',
    'charset'  => 'utf8'
];
```
