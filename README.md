## Installing CMS

```
git clone https://github.com/developer01234/CMS-PHP.git

composer update

php -S localhost:8000
```

Example /config/database.php

```
<?php
return [
    'host'     => 'localhost',
    'db_name'  => 'cms',
    'username' => 'root',
    'password' => '',
    'charset'  => 'utf8'
];
```
